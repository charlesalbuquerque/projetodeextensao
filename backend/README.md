# Projeto de Extensão
A ideia do nosso projeto é criar uma aplicação que seja simples, ágil e eficiente. Nela, qualquer pessoa poderá registrar a localização e a situação de um animal de rua, criando um alerta em tempo real. Além do sistema de alertas, o aplicativo contará com um mural de adoção, informações sobre castração, posse responsável e contatos de emergência.
